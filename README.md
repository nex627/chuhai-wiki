## 三个差的底层逻辑

- **文化差**：中国人觉得不值钱的东西老外觉得稀罕 → 独立站+Adsense（被动）
- **技能差**：AI降低了技能门槛 → Fiverr接单（主动）
- **平台差**：海外平台有中国人不知道的赚钱方式 → 联盟营销（半被动）
- 文化差是中国人唯一的天然护城河

## AI嵌入模式

AI不是独立项目，是每个项目的80%劳动力。核心原则="AI做80%你做20%"——你只负责选品决策+质量审核+收款

## 如何使用

1. 按类型浏览 `wiki/` 目录下的分类文件
2. 搜索关键词找到相关项目
3. 用AI调用本知识库做项目筛选和方案生成
4. 读取 `cards.json` 进行程序化检索
